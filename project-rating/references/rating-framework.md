# 评级框架参考

## 评估维度与门槛

### ① 基本面 & 产品

| 等级 | 条件 |
|------|------|
| 🟢 A 级 | 满足以下 **2 项**以上：主网/测试网落地且创新（或 L1/复杂基建）；DAU>10K 或 MAU>500K；TVL>$100M；7D Fees>$1M 或 周收入>$100K |
| 🟡 B+ 级 | 满足以下 **2 项**以上：主网/测试网已上线；DAU>1K 或 MAU>100K；TVL>$20M；7D Fees>$10K |
| 🟠 B- 级 | 产品逻辑通顺，有产品落地，有一定真实用户或刷量基础数据，非纯概念 |

### ② 赛道风口

| 等级 | 条件 |
|------|------|
| 🟢 A 级 | 核心叙事：基建、DeFi（重点 Perp DEX）、PayFi、高频 Consumer App、RWA、预测市场 |
| 🟡 B+ 级 | 热门叙事的"二龙/三龙"（如新兴公链头部），或具备独特资源优势；或涵盖上述热门叙事 |
| 🟠 B- 级 | 视具体赛道，无强制硬性要求 |

### ③ 融资规模

| 等级 | 条件 |
|------|------|
| 🟢 A 级 | $10M+ 且 2 家以上头部 A 级机构，或 1 家 A+ 级资本 |
| 🟡 B+ 级 | $2M–$10M，且 1 家 Tier 2+ 或 2+ 家中等 B- 以上资本 |
| 🟠 B- 级 | 融资金额/资方已披露，RootData 平台可查 |

### ④ 团队背景（加分项，非强制硬性要求）

| 条件 | 说明 |
|------|------|
| A 级加分 | 顶级公链/机构强绑定孵化或背书；全球知名人物/Web2 巨头背书；有已 TGE 龙头项目核心成员 |
| 基础要求 | 非匿名团队（非强制） |

### ⑤ 社群热度

| 等级 | 条件（满足其二） |
|------|------|
| 🟢 A 级 | 推特粉丝 50K+；5+ 头部 KOL（50K+）且含 2+ 中英文 KOL（50K+）讨论；TGE 前持续 2 个月以上自发讨论 |
| 🟡 B+ 级 | 推特粉丝 10K+；局部热度高，3+ KOL（10K+）讨论；TGE 前持续 15 天以上自发讨论 |
| 🟠 B- 级 | 推特粉丝 10K+；5 位以上 1K+ 中低腰 KOL 讨论；市场有非刷量真实讨论 |

### ⑥ 交易所预期

| 等级 | 条件（满足其一） |
|------|------|
| 🟢 A 级 | 明确/高概率上币：Coinlist/Legion 等头部打新；Coinbase/Bybit/Upbit/Bitget；Binance Alpha/合约，以上两家起 |
| 🟡 B+ 级 | 高概率上币：Binance Alpha / Bybit (+Bitget) / Upbit（现货），满足一项 |
| 🟠 B- 级 | 1 家第二梯队交易所；或 2 家以上第三梯队交易所预期 |

## 综合评级判定规则

| 综合评级 | 判定门槛 |
|---------|---------|
| 🟢 A 级 | 满足上述维度 **4 项** |
| 🟢 A- 级 | 满足上述维度 **3 项** |
| 🟡 B+ 级 | 满足上述维度 **3 项以上**（B+ 门槛） |
| 🟠 B- 级 | 满足上述任意 **1 项** |
